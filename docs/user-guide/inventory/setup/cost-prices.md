#Cost Prices

Cost Price is used to specify the cost price of an item. The cost price set in
this overrides the cost price of item in <a href="item-maintenance.md">Item Maintenance</a>.

##Fields

**Item Cost Price Id**

This is an auto-generated and read-only field.


**Item Id**

Select the item or search by clicking the icon <i class="item-selector"></i>.
View <a href="item-maintenance.md">Item Maintenance</a> for more information.

**Unit Id**

Choose the unit of measurement. The cost price is set for the unit selected
or search by clicking the icon <i class="item-selector"></i>.
View <a href="uom.md">Unit of Measurement</a> for more information.

**Party Id**

Select the party if you want specific parties to have the cost prices. 
View <a href="party-accounts.md">Party Accounts</a> for detailed information.

**Lead Time in Days**

A business which sells tangible goods has suppliers. 
Lead Time denotes how long will the supplier take to provide goods to the business.
Enter the lead time in days.

**Price**

Specify the cost price of the selected unit of item.
